Installer Bash et CLang sur son ordinateur

Plutôt que de passer par Turing, vous pouvez installer Bash et CLang directement sur votre machine. Selon votre système d'exploitation, la procédure ne sera pas tout à fait la même.

Si un de ces guide ne fonctionne pas comme prévu, vous pouvez demander de l'aide en envoyant un mail à webmaster arobase edgarogh point fr.

Mac OS X et Linux

Premièrement, vous aurez besoin d'ouvrir un terminal.

  • Sous Mac OS X, recherchez « Terminal » dans la liste des applications, et prenez celle pré-installée.
  • Sous Ubuntu, ouvrez le menu « Applications », et cherchez « Terminal ». Prenez là aussi l'application déjà présente.
  • Si vous avez une autre distribution Linux, vous savez probablement déjà faire (si ce n'est pas le cas vous pouvez envoyer un mail à ana arobase gelez point xyz pour de l'aide).

Ce terminal permet d'exécuter des commandes Bash sur votre ordinateur, si vous voulez seulement faire du Bash il sera donc suffisant. Pour installer CLang :

  • Sous Mac OS X, ouvrez l'App Store, et installez Xcode. C'est l'éditeur de code d'Apple qui permet entre autre de faire du C, et qui fournit CLang.
  • Sous Ubuntu, Debian, Linux Mint, elementary OS, etc : tapez sudo apt install -y clang, et entrez votre mot de passe.
  • Sous Fedora, RHEL ou CentOS, tapez sudo dnf install -y clang et entrez votre mot de passe.

CLang est normalement maintenant installé sur votre ordinateur. Vous pouvez vérifier en entrant la commande clang --version dans votre terminal, qui vous affichera la version de CLang si il a été installé correctment.

Windows

Téléchargez et lancez l'installateur de CLang. Pour ouvrir un terminal, faites Windows+R, tapez « powershell » dans la fenêtre qui s'ouvre, et validez. Vous pouvez normalement lancer CLang.
