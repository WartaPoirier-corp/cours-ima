Se connecter à Turing en SSH

Les serveurs de Turing sont ceux mis en place par l'Université. Ils ont tous les logiciels dont nous avons besoin d'installés (Bash et CLang principalement). On peut y accéder grâce au protocole SSH (c'est ce qu'on fait en TP). Selon votre système d'exploitation, la connexion SSH se fait de façon différente.

Mac OS X et Linux

Premièrement, vous aurez besoin d'ouvrir un terminal.

  • Sous Mac OS X, recherchez « Terminal » dans la liste des applications, et prenez celle pré-installée.
  • Sous Ubuntu, ouvrez le menu « Applications », et cherchez « Terminal ». Prenez là aussi l'application déjà présente.
  • Si vous avez une autre distribution Linux, vous savez probablement déjà faire (si ce n'est pas le cas vous pouvez envoyer un mail à webmaster arobase edgarogh point fr pour de l'aide).

Ce terminal permet d'exécuter des commandes Bash sur votre ordinateur, si vous voulez seulement faire du Bash il sera donc suffisant (pour du C, il faudra installer CLang). Mais si vous voulez vous connecter à Turing pour ne rien installer en plus sur votre ordinateur, vous pouvez utiliser la commande ssh : 

ssh NOM@turing.e.ujf-grenoble.fr

Il faudra remplacer NOM par votre identifiant Agalan (celui que vous utiliser sur Léo, Chamilo, Caséine, etc). Faites Entrée pour lancer la commande. Votre mot de passe sera demandé : là aussi c'est le mot de passe Agalan. Aucun caractère ne s'affiche, c'est normal, c'est pour plus de sécurité. Faites Entrée quand vous avez fini de le taper, vous êtes connecté⋅e !

Windows

Ce tutoriel n'a pas été testé et se base sur le guide fourni par le professeur.

Téléchargez et installez MobaXterm à cette adresse. Une fois que c'est fait, lancez l'application.

Cliquez sur « Start Local Terminal »

Cliquez sur le bouton « Session », puis sur le bouton « SSH » Dans la case « Remote host * », entrez l’adresse du serveur : turing.e.ujf-grenoble.fr. Cocher la case « Specify username », et entrez votre identifiant Agalan (celui utilisé sur Léo, Chamilo, Caséine, etc). Cliquez sur OK.

Entrez votre mot de passe. Aucun caractère ne s'affiche mais c'est normal, c'est pour plus de sécurité. Vous avez à présent accès au serveur Turing. Vous pouvez entrer des lignes de commande dans le terminal.

Pour modifier vos fichiers, au lieu d’utiliser le logiciel gedit, vous pouvez ouvrir les fichiers en parcourant l’arborescence située à gauche de la fenêtre. Les applications graphiques pourraient en effet ne pas marcher ou être très lentes à cause de la distance entre votre machine et le serveur sur le campus.
